# JMJ Fabrication Works — Image Folder

Upload your project photos into the relevant subfolder:

## 📁 Folder Structure

| Folder | What to upload |
|---|---|
| `gallery/` | General project photos (homepage gallery) |
| `tensile/` | Tensile structure photos |
| `parking/` | Car parking shed photos |
| `structural/` | Structural steel fabrication photos |
| `awnings/` | Awning and canopy photos |

## 📌 Tips
- JPG or PNG format preferred
- Recommended size: 1200×800px or larger
- Keep file names simple: `project-01.jpg`, `school-tensile.jpg` etc.
- After uploading, tell your developer and the website will be updated!
